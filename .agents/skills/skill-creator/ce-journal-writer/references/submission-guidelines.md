# 计算机教育期刊投稿指南

## 1. 投稿程序

### 1）在线投稿流程
- 首先点击"作者在线投稿"，进入后请仔细阅读投稿说明
- 注册账号，使用注册后的登录名、密码进入投稿系统
- 点击"我要投稿"，仔细阅读投稿须知、论文模板、著作权转让合同
- 按照投稿向导进行投稿：
  - 上传稿件全文
  - 上传非涉密证明（简要说明内容非涉密，落款单位名、日期并加盖学院或学校公章）
  - 点击"下一步"，继续完成后续投稿步骤
- 投稿后请耐心等待，不要重复投稿
- 审稿周期为一个月，录用后出版周期大约半年

### 2）投稿后注意事项
- 请下载论文模板并据此规范稿件
- 投稿后请耐心等待，可通过首页的"作者在线查稿"随时查看稿件处理情况
- 如无特殊情况，编辑部不接受电话查询

### 3）修改稿提交
- 作者收到审稿专家的修改意见后，请认真修改并撰写修改说明
- 上传修改稿需点击"投修改稿"

## 2. 来稿要求和注意事项

### 1）基本要求
- 来稿务必重点突出，论据充分，数据可靠，文字简练
- 每篇论文（含图、表、参考文献、英文摘要）一般不超过6000字
  - 有创新性的论文可适当放宽要求，但要突出创新点
- 投稿同时必须提交著作权转让合同（PDF）
- 论文必须包括（按顺序）：题目、摘要、关键词（3～8个）、分类号、正文、参考文献
- 中文题名不超过20个汉字
- 文章格式参见论文模板

### 2）摘要要求
- 应尽量写成报道性摘要，包括研究的目的、方法、结果和结论
- 切忌把应在引言中出现的内容写入摘要
- 不要对论文内容作自我评价
- 采用第3人称的写法，不要使用"本文"、"作者"等作主语
- 建议采用"为了（针对，根据）......对......进行研究（分析）"、"阐述……过程，提出......观点"等表述
- 字数以200～300字左右为宜

### 3）格式规范
- 文稿用5号宋体
- 量和单位符号等必须符合国家标准的规定
- 稿中外文字母、符号必须分清大、小写、正、斜体
- 上、下角的字母、数码和符号要清晰
- 矩阵、矢量用黑体表示

### 4）图表要求
- 文中图、表只附最必要的
- 凡能用文字说明的尽量不用图表

### 5）稿件处理
- 来稿如不符合上述要求，编辑部将先退请作者修改后再送审
