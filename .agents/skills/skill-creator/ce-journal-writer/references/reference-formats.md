# 参考文献格式规范

## 基本要求
只选最主要的、发表在正式出版物上的文献。内部文件和资料不能作为参考文献引用。无特殊需要不必罗列一般知识性或陈旧史料。文献序号按其在文中出现先后次序编排。作者姓名用外文的，应按中文习惯，姓前名后，名可缩写，去掉缩写点，只留空。3人以上作者只列3人，后加"等"字。

## 各类文献格式

### 期刊论文
```
[序号] 作者. 题名[J]. 期刊名称(外文可缩写), 出版年份, 卷号(期号): 起止页码.
```

**示例：**
```
[1] 张三. AI赋能教育的实践与思考[J]. 计算机教育, 2023, 20(5): 12-18.
[2] Smith J, Johnson M. AI in Education Research[J]. J. Educ. Tech., 2023, 15(2): 45-52.
```

### 专著
```
[序号] 作者. 书名[M]. 出版地: 出版者, 出版年. 起止页码.
```

**示例：**
```
[3] 李四. 人工智能与教学改革[M]. 北京: 高等教育出版社, 2022. 120-135.
[4] Wang L. AI in Modern Education[M]. New York: Springer, 2023. 88-102.
```

### 论文集
```
[序号] 作者. 题名[C]//论文集主编者. 论文集名. 出版地: 出版者, 出版年: 起止页码.
```

**示例：**
```
[5] 王五. 大模型驱动的教学变革[C]//李明. 计算机教育创新文集. 北京: 清华大学出版社, 2023: 234-245.
[6] Chen X. AI Applications in Learning[C]//Zhang Y. Advances in Educational Technology. Singapore: World Scientific, 2023: 156-168.
```

### 学位论文
```
[序号] 作者. 题名[D]. 保存地点: 保存单位, 年份.
```

**示例：**
```
[7] 赵六. AI赋能计算机课程思政建设研究[D]. 北京: 北京大学, 2023.
[8] Davis K. AI Ethics in Higher Education[D]. Stanford: Stanford University, 2023.
```

### 标准
```
[序号] 主要责任者. 标准编号—发布年, 标准名称[S]. 出版地: 出版者, 出版年: 页码.
```

**示例：**
```
[9] 国家质量监督检验检疫总局. GB/T 7714—2015, 信息与文献 参考文献著录规则[S]. 北京: 中国标准出版社, 2015: 10-15.
```

### 电子文献
```
主要责任者. 电子文献题名[EB/OL]. 出版地: 出版者, 出版年(更新或修改日期)[引用日期]. 获取或访问路径.
```

**示例：**
```
[10] 教育部. 关于加强高等学校在线开放课程建设应用的管理办法[EB/OL]. 北京: 教育部, 2023-05-15[2023-12-01]. http://www.moe.gov.cn/srcsite/A08/s7056/202305/t20230515_xxxxx.html.
```

### 未定义类型文献
```
作者. 文献题名［Z］. 出版地: 出版者, 出版年.
```

**示例：**
```
[11] 刘七. 教育数字化转型的挑战与机遇[Z]. 北京: 教育科学出版社, 2023.
```

## 注意事项

1. **序号格式**：使用方括号`[序号]`，文中引用应为`[1]`、`[2]`等
2. **作者格式**：
   - 中文：姓在前，名在后，如"张三"
   - 外文：姓在前，名缩写在后，如"Smith J"
   - 3人以上只列前3人，加"等"，如"张三, 李四, 王五, 等"
3. **标点符号**：
   - 作者后用点`.`，题名后用方括号`[J]`等
   - 期刊后用逗号`,`，出版年份后用逗号`
   - 期号用圆括号`( )`，页码用冒号`:`
4. **文献标识**：
   - 期刊论文：`[J]`
   - 专著：`[M]`
   - 论文集：`[C]`
   - 学位论文：`[D]`
   - 标准：`[S]`
   - 电子文献：`[EB/OL]`
   - 其他：`[Z]`
