# 本科生教育-微信群课堂交流数据

## 背景

针对本科生教育面临的三大挑战：
- **因材施教难**：难以兼顾不同基础学生的需求
- **教学反馈难**：无法及时掌握学生学习状况
- **结伴学习难**：缺乏有效的互助学习机制

## 方案

每次上课将全班学生随机分为12组并建立微信群，通过课堂讨论与课后交流，利用大模型分析学生对话内容，实现以下目标：

### 分析维度

| 维度 | 类别 | 说明 |
|------|------|------|
| **学生分类** | 学有余力 | 需要更高难度的拓展 |
| | 学习困难 | 需要针对性辅导 |
| | 参与度低 | 需要激励和引导 |
| **知识点掌握** | 良好 | 理解准确，能运用 |
| | 不足 | 存在认知误区 |
| **关系网络** | 结伴学习 | 识别学习伙伴关系 |
| | 一对一帮扶 | 匹配互助对象 |

## 数据说明

- 覆盖**6次课程**的学生微信群交流记录
- 每个**txt文件**对应一次课程中一个小组的完整对话记录
